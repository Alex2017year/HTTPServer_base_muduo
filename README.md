
# HTTPServer_base_muduo


一个基于[muduo](https://github.com/chenshuo/muduo)的项目，用于学习和研究



#### Feature:
  - C++11
  - non-blocking IO + IO multiplexing(Reactor)
  - multithreading
  - timer

#### Target
- 在muduo基础上，用c++11替换原有的boost组件，使用c++11新特性
- 尝试添加epoll ET模式
- 增加HTTP的功能，用php-fpm支持php
